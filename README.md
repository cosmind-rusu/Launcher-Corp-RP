# Launcher Corp

Un Launcher basado en el lenguaje C#, con funciones de conexion a FiveM.

Desarrollado por Cosmin con mucho ❤ para FireRP


## Instalación

Para instalar el launcher seguiremos los siguientes pasos:

```bash
  Releases
  Ultima version
  Descargar launcherCorp.exe
  Ejecutarlo
```
